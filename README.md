## Principio de continuidad

El repositorio es la memoria oficial del proyecto. Una persona o una IA debe poder retomarlo leyendo los documentos rectores, sin depender del historial completo de conversaciones.

## Versionado

- Una ciudad o hito editorial cerrado incrementa la **versión mayor**.
- Una sesión o incorporación relevante incrementa la **versión menor**.

Ejemplo: `2.3` identifica la tercera evolución relevante dentro del hito editorial 2.
